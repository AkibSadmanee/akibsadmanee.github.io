Selected Fun Projects

Kani - LLM driven chatbot

# 3rd place in Aloha Data AI Hackathon 2025
Used the LLM Agentic Programming (highly constrained with tools and functions) along with chain-of-thought prompting to act as a helper chatbot for the Hawaii Climate Data Portal.

Mailfinder

A Python-based webscraping project. Googles a given topic and scrapes the first n (user choice) pages of results to find all the email addresses and verify them using a 3rd party package.

Shobdokutir-tacotron

A pipeline for Bangla Text-to-Speech system using the Waveglow model and NVIDIA's Tacotron model.

ParrotForms

Dynamically and automatically fill medical forms from conversations using AI, saving hours and potentially even lives.

Pokemon GBA Bot Behavioral Cloning

A bot to level up Pokémon automatically in Pokémon FireRed version using behavioral cloning.

Bionic Reader

A Python-based bionic reader software that helps you read faster if you are a slow reader.

Bijoy to Unicode Converter

A Python project to convert Bangla text from an old mapped styling to Unicode styling.

Image Processing Functions

Self-written Matlab functions to perform image processing tasks.
